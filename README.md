# ProcessVoice Diagrams

Process diagrams created with [ProcessVoice](https://processvoice.com).

Each process lives in `processes/<slug>/` as:
- `<slug>.d2` — D2 diagram source (text, version-controlled)
- `<slug>.svg` — rendered diagram (auto-generated by GitHub Action)
- `metadata.json` — title, description, input type, timestamps

The commit history on each `.d2` file is your audit trail — every version of every process, who changed it, and when.
